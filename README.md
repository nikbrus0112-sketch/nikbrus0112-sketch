## What I'm building

Working through Andrej Karpathy's *Neural Networks: Zero to Hero* series and 
beyond, implementing everything from scratch to actually understand what's 
happening under the hood rather than just calling library functions:

- **[micrograd](https://github.com/nikbrus0112-sketch/micrograd)** — a tiny 
  autograd engine, backpropagation implemented from first principles
- **[makemore](https://github.com/nikbrus0112-sketch/makemore)** — character-level 
  language models, from a bigram model up through a WaveNet-style architecture, 
  with a hand-tuned MLP that beat the reference implementation's loss (2.011 vs 2.17)
- **[nanoGPT](https://github.com/nikbrus0112-sketch/nanogpt)** — a GPT-style 
  transformer built from scratch, implementing self-attention and the full 
  architecture without pre-built libraries
- **[encoder](https://github.com/nikbrus0112-sketch/encoder)** — a byte-pair 
  encoding tokenizer, including a basic version implemented independently 
  from first principles before seeing any reference code
